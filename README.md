### 作品简介

#### 1. P5R — 关于《女神异闻录5 皇家版》的游戏批评
以《Persona 5 Royal》为对象，从游戏机制、叙事结构与意识形态三个维度展开理论分析，探讨 JRPG 如何在"日常—非日常"的双重时间性中构建玩家主体性，并对游戏中的社会批判维度进行再批判。

#### 2. 精灵式生活指南（暂定） — 精灵/长生种想象力的文化批评
从托尔金的精灵形象出发，横跨奇幻文学、ACG 与当代网络文化，考察"长生种"这一想象装置如何折射人类对时间、死亡与生活方式的焦虑与渴望。标题中的"暂定"本身即是对写作过程开放性的一种姿态。

#### 3. 森见登美彦（完成） — 森见登美彦文学评论
系统梳理森见登美彦的创作谱系，从《太阳之塔》到《夜行》，分析其独特的"京都妄想"文体如何在宅文化与纯文学之间架设桥梁，以及其作品中反复出现的"拟似家族"主题的深层文化逻辑。

#### 4. gquuuuuux — 高达系列（GQuuuuuuX）动画批评
聚焦《机动战士高达》系列最新作《GQuuuuuuX》，在高达宇宙的宏大谱系中审视这部备受争议的新作，探讨其在机甲动画类型传统与当代审美转向之间的张力。

#### 5. 在这巨人消失的时代，要来一盒理论预制菜吗？ — 后现代知识生产随笔
以犀利而自反的笔触，诊断当代知识生产中的"理论预制菜"现象——当宏大叙事退场，理论是否沦为即食速消费品？一篇兼具智识密度与阅读快感的时代诊断书。

#### 6. 欲加之罪，何患无辞？ — 托洛茨基研究学术辩护
全文约 31,000 字，是仓库中篇幅最长的作品。以严谨的史学方法回应对托洛茨基研究的种种质疑与曲解，既是学术辩护，也是一堂关于如何"在敌意中做研究"的方法论示范。
